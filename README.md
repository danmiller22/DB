# US Team Fleet — Single-file Dashboard

Static glass UI dashboard that reads data from your Google Sheet.

## Deploy on GitHub Pages
1. Create a repository and add **index.html** from this zip.
2. Enable **Settings → Pages → Deploy from a branch → main / root**.
3. Make sure your Sheet is public for viewing or publish to web (CSV).
